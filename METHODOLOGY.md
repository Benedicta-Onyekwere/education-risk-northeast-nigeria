Methodology NoteEducation Risk Mapping — Northeast NigeriaThis note documents all data sourcing, cleaning decisions, analytical logic, and replication steps for the EBI conflict-sensitive education planning tool. It is intended for technical reviewers, future implementers, and EBI staff who wish to adapt this methodology to other country contexts.1. Data Sources and Selection RationaleFour open, publicly available datasets were used:Conflict events: UCDP (Uppsala Conflict Data Program) georeferenced event dataset for Nigeria, filtered to Borno, Yobe, and Adamawa, 2023–2024. Available at ucdp.uu.se and via HDX. UCDP was selected over ACLED because it provides a complete event-level CSV without requiring registration, keeping the project fully open-source and reproducible without access barriers.School locations: HOT-OSM verified school dataset for Nigeria, downloaded from HDX (data.humdata.org). The GRID3 Nigeria Education Facilities dataset was initially considered as it is the most complete ground-truthed national dataset including school types and operational status. However the HOT-OSM dataset was used for the final verified records as it provided better coordinate accuracy for the northeast states.Population estimates: UNFPA Nigeria Subnational Population Statistics 2020, downloaded from HDX (data.humdata.org/dataset/cod-ps-nga). The nga_admpop_2020.xlsx file was used, specifically the nga_admpop_adm2_2020 sheet which provides LGA-level sex and age disaggregated population figures. WorldPop gridded rasters were initially considered but the UNFPA COD-PS dataset was selected as it provides LGA-level figures already aggregated and validated by the UN, making it directly usable without raster processing. The 5–14 age band provides granular estimates of school-age children rather than relying on outdated census totals.Administrative boundaries: LGA boundary GeoJSON for Nigeria (nga_adm2_boundaries.geojson), sourced from OCHA. The LGA is the primary unit of analysis because it is the primary planning unit for both EBI and the Nigerian government.2. Data Pipeline OverviewThe analysis followed a Define → Find → Get → Inspect → Clean → Analyse → Visualise pipeline:Define: Core question: which LGAs in Borno, Yobe, and Adamawa have the highest concentrations of school-age children exposed to active conflict with insufficient school access?Find: Identified UCDP, HOT-OSM, UNFPA COD-PS, and OCHA boundary datasets as open, citable sources covering the target geography.Get: Downloaded each dataset and filtered to the three target states using Python and Pandas scripts. Geographic filtering used state codes BR (Borno), YO (Yobe), and AD (Adamawa). Temporal filtering restricted conflict events to a 24-month analysis window.Inspect: Conducted initial quality checks on each file, identifying missing coordinates, empty columns, and data integrity issues before cleaning.Clean: Applied systematic cleaning to each dataset as documented in Sections 3–5 below.Analyse: Calculated composite risk scores per LGA and produced pivot table summaries of conflict type, fatality distribution, and district rankings.Visualise: Built interactive React and Leaflet.js dashboard published on GitHub Pages.3. Conflict Data Cleaning (northeast_conflict.csv)The UCDP dataset required minimal cleaning:Filtered to Borno, Yobe, and Adamawa states and the 24-month analysis window, producing 265 conflict events.All 265 retained records had complete latitude and longitude coordinates. No records were removed for missing coordinates.The gwnob column was deleted as it contained no values across all 265 rows.The gwnoa column retained 59 blank values. This column contains a numeric country code not used in spatial analysis. Blanks were left as-is.The source_original column retained 154 blank values reflecting common absence of direct URL citations. Not used in analysis.The adm_2 column retained 39 blank values for events recorded without a specific LGA assignment. Since analysis uses coordinates for spatial joining, these blanks do not affect results.The best column (fatality best estimate) had zero blank values. No imputation required.Final dataset: 265 conflict events, 2,071 total fatalities.Breakdown by violence type:Type 1 (State-based violence): 206 events, 1,208 fatalitiesType 2 (Non-state conflict): 19 events, 586 fatalitiesType 3 (One-sided violence — civilians directly targeted): 40 events, 277 fatalitiesThe 40 civilian-targeting incidents confirm communities face deliberate targeting, not only incidental exposure to conflict between armed actors. This is the most analytically significant finding for EBI’s education mandate.4. School Location Data Cleaning (verified_northeast_schools_cleaned.csv)The school dataset required extensive manual cleaning. The original HOT-OSM download contained 167 records. After full record-by-record review in Google Sheets, 91 records were removed and 76 verified education facilities were retained.4.1 Records removed and reasonsNigeria SE4All source records: Records carrying the source tag nigeriase4all.gov.ng were removed. Nigeria SE4All is the Sustainable Energy for All programme — an electricity access initiative whose records were incorrectly tagged as school amenities in OpenStreetMap. These records had no school names, no addresses, and no humanitarian verification. The source column was retained in the dataset to document this provenance decision.Individual rooms, classrooms, and building blocks: Records representing individual physical spaces within a single institution were removed, including Lecture Theatre A&amp;B, Geography Dept Room 3&amp;4, FCE(T) Administration, Class 5A, Classroom 1A, B Block, and Block C. These reflect over-mapping of single campuses in OpenStreetMap where every room was tagged as a separate school record.Non-education facilities: Records identified as non-education facilities were removed: restroom (male and female), Mosque Road (a street landmark), SRC Secretariat Gombe State University (student union office), Senate Chamber Gombe State University (administrative meeting room), Gombe State SRC Park (recreational space), and Dr Abubakar Abdullahi Baba Jolly Phonics Resources Room (single named room within a larger institution).Standalone personal names: Records consisting only of a personal name with no institutional descriptor were removed: Musa Babayo, Rufai Ahmad Alkali, Ibrahim Waziri Abubakar, Haruna Kadir Rashid, Sen Joshua Lidani, and JWD Somasundara. The last record was also flagged as potentially geocoded to the wrong region.Academic departments within universities: Faculty-level subdivisions of universities (nursing department, biochemistry department, geography department) were removed as they represent parts of a single institution rather than distinct schools. Properly constituted schools with institutional standing — School of Health, School of Women Primary-Secondary — were retained.4.2 Records retainedThe following categories were retained regardless of naming convention:Standard school identifiers: Primary School, Secondary School, JSS, SSS, Government School, Community SchoolIslamic schools with Madarasa, Madarasatu, or Makaranta prefixes — including Madarasatu Alarama Mallam Gambo Littah Fizzul Quran, a Quranic school following standard northern Nigerian naming conventionsRaudha records — Islamic early childhood education centresUniversal Basic Education (UBE) facilitiesNafsts Buni Yadi — retained as a locally known institution in Buni Yadi, Yobe State, a location of particular significance given the 2014 Boko Haram attack on its schoolGombe State University Daycare — legitimate early childhood education facilityAdamawa Agricultural Development Programme — community vocational education institution4.3 Duplicate name checkA conditional formatting check in Google Sheets identified two schools sharing the name Mara Imul Iman and two sharing Bubayero Primary School. Both pairs had different coordinates and OSM IDs confirming distinct physical locations. Both pairs were retained.4.4 Data correctionsOne record (ZBA) had a missing amenity value and was manually coded as school based on the building field value and a named local contributor source (Mr Auwal Abdallah), consistent with OpenStreetMap ground-truthing standards. One record had the school name (Idi primary school) incorrectly placed in the name_en column rather than the name column — this was corrected manually.4.5 Column managementThe name_en column was deleted as only one of 167 rows contained a value. The capacity_p column was deleted as it contained no values across the entire dataset. The source column was retained despite sparse values as it documents provenance for key exclusion decisions. All column headers were standardised to uppercase.Final dataset: 76 verified education facilities. Amenity breakdown: 67 schools, 5 kindergartens, 3 colleges, 1 university.5. Population Data Processing (nga_admpop_2020.xlsx)The UNFPA Nigeria Subnational Population Statistics 2020 file was downloaded from HDX (data.humdata.org/dataset/cod-ps-nga). The nga_admpop_adm2_2020 sheet was used, which provides LGA-level sex and age disaggregated population projections.The 5–14 age band was calculated by summing the T_05_09 and T_10_14 columns (total male and female combined) for each LGA. Records were filtered to Borno, Yobe, and Adamawa states, producing population figures for 65 LGAs. These were merged with the risk scores file using LGA name as the join key.Population figures for the top five risk districts:Kukawa (Borno): 107,385 children aged 5–14Gwoza (Borno): 125,905 children aged 5–14Marte (Borno): 68,283 children aged 5–14Abadam (Borno): 45,707 children aged 5–14Bama (Borno): 86,109 children aged 5–14Note: Gwoza has a larger school-age population than Kukawa despite a lower risk score. This reveals two distinct crisis profiles: Kukawa represents extreme conflict lethality (601 fatalities from 37 events), while Gwoza represents the largest scale of child vulnerability (125,905 children in a sustained conflict zone). Both require immediate intervention but for different programmatic reasons — a distinction only visible when population data is included alongside conflict intensity.Data limitation: These figures are 2020 projections based on the 2006 Census, the most recent available LGA-disaggregated dataset from this source and the standard reference used by UN agencies operating in Nigeria. Actual 2024 figures may differ due to displacement caused by ongoing conflict.6. LGA Risk Score Calculation (northeast_lga_risk_scores_cleaned.csv)A composite risk score was calculated for each LGA using the following weighted formula:35% weight: Conflict event frequency (number of UCDP events in the analysis period)45% weight: Fatality severity (total fatalities using UCDP best estimate field)20% weight: Population vulnerability (school-age children aged 5–14 per LGA)The 45% weighting for fatality severity reflects the judgement that lethality is the strongest indicator of education access disruption. This is confirmed by the data: Kukawa scores 10.0 with 37 events and 601 fatalities, while Gwoza scores 4.4 despite 53 events, because its fatality total is significantly lower per event.Scores were normalised to a 0–10 scale. LGAs with no recorded conflict events were assigned 0.0, reflecting absence of recorded incidents rather than confirmed safety — underreporting is common in remote areas of northeastern Nigeria.6.1 Sensitivity checkA sensitivity check comparing 12-month and 36-month analysis windows was conducted. The 24-month window was selected as it is long enough to capture seasonal conflict patterns while excluding the 2014–2016 peak insurgency years that would distort the picture toward areas that have since stabilised.6.2 CoverageThe final dataset covers 104 LGAs: 26 with active risk scores above zero and 78 with scores of 0.0. The three states contain 112 LGAs total; 8 could not be matched with reliable centroid coordinates and were excluded. Green circles on the map represent the zero-score districts — relatively stable areas that may be hosting displaced populations from high-risk zones.7. Replication in a New Country ContextDownload UCDP data filtered to the target country and relevant states. Filter to a 24-month window as a starting point.Download the HOT-OSM school dataset for the target country from HDX. Conduct manual record-by-record review to remove non-school entries. Document all exclusion decisions.Download the UNFPA COD-PS population dataset for the target country from HDX. Extract the 5–14 age band from the adm2-level sheet.Obtain LGA or district boundary GeoJSON from HDX or GADM.Run the filtering and risk score scripts in the repository, substituting new country data files. Adjust weighting formula if the conflict context warrants different balance.Deploy updated data files to GitHub repository and rebuild the map using the existing codebase.10.
